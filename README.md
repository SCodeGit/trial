# 📘 SCode Coepasco Online – Colleges of Education Past Questions

☀️ Empowering Learning Through Past Questions

Welcome to **SCode Coepasco Online**, a digital platform designed to make learning easier for trainees in Colleges of Education.  
Here, students can access organized past questions in PDF format by selecting their **University, Level, Semester, and Program**.

---

## 🚀 Features
- 📂 Access to past questions in structured PDF format
- 🔍 Search functionality for quick retrieval
- 🎓 Filter by University, Level, Semester, and Program
- 📱 Responsive design for mobile and desktop
- 🔒 Secure hosting with privacy and terms of use

---

## 📖 Usage
1. Visit [www.scode-coepasco.online](http://www.scode-coepasco.online)  
2. Select your **University, Level, Semester, and Program**  
3. Browse and download available **PDF past questions**  
4. Use the search bar to quickly locate specific materials  

---

## 📂 Project Structure
- `/public` → Static assets (CSS, JS, images)   
- `/docs` → Instructions and trainee guides  
- `/pdfs` → Past questions organized by category
